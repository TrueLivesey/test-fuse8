# Тестовое задание для Fuse8. Павлов Никита

## Описание

Сайт для поиска персонажей из сериала "Рик и Морти".

## Как развернуть проект:

- Установить зависимости **npm i**
- Запустить сборку командой **npm run dev**

## Технологии

- React
- Vite
- Axios
- React Query
- Lodash
- PropTypes
- SCSS
- PostCSS
